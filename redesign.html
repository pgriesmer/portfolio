Brown CS Website Redesign

CS1300 Project, Fall 2018

Overview

As part of my UI/UX course this fall, I redesigned the website for Brown's Computer Science Department.

The Brown CS Department's website is plagued by inconsistent, disorganized layout and suffers from a generally unprofessional look overall. Classmates of mine have described the color scheme and fonts as "gimmicky," and the navigation presents users with more options than it should. The media queries it uses often make the site look worse at small screen sizes.

Current homepage
Redesigned homepage

1 // Lo-Fi Mockups

I began my project by doing lo-fi mockups for new versions of five preexisting pages for tablet-sized screens.

Home Events Rooms Industry Partners Undergraduate Degree

My redesign created consistency from page to page by introducing a three-column layout. Whitespace was added to make the site much easier on the eyes. The site's main navigation menu was simplified significantly and breadcrumbs were made to appear on every page (rather than on certain pages and not on others, as is currently the case). Interactive tools (e.g., calendars) and photos were added throughout to make the site more engaging and easier to understand on certain pages (e.g., the "Events" page).

2 // Hi-Fi Prototype

Once I had finished the lo-fi mockups shown above and settled on a general style for the website, I did a hi-fi prototype for the site's homepage.

The fonts were chosen to give the site more gravitas, and colors were chosen from a very simple palette to give the site a much more understated appearance (and generally go unnoticed by the user). Ideally, photos on the homepage would update dynamically based on the latest "News"/"Blog" articles and "Events" blurbs.

3 // Responsive Site

I then coded up a responsive version of the homepage where I made a number of adjustments for mobile screens and 4K screens (in case the site was ever used in a presentation).

On mobile, a one-column layout replaced the default three-column layout for readability, and the navigation menu was simplified even further by replacing it with a hamburger icon.

Mobile

The 4K version of the site featured the last two combined columns of the tablet/desktop view broken out into two separate columns, so that the images on the site would appear in reasonable proportion to one another.

4K

Outcomes

This project was the first time I had ever (1) done a series of lo-fi mockups for the same site and (2) designed a hi-fi prototype based on a lo-fi design. It also taught me how to use Balsamiq for lo-fi designs and Adobe XD for hi-fi designs.

If I had more time to work on the project, I would have scaled down the hero image that appears atop the homepage so that users wouldn't have to scroll as far down to get to the content beneath. I would also make this same image into a carousel and would have resized several fonts on the 4K version of the site to make them appear proportional to the rest of the content on the page.
